# Dicas do Terminal

Vou listar alguns comandos que estamos aprendendo:

- `cd` (navega enter pastas)
Exemplo para entrar dentro de uma pasta
```
cd nomeDaPasta
```

Exemplo para sair de uma pasta:
```
cd ..
```

- `mkdir` (criar pastar)
Exemplo de como criar uma pasta
```
mkdir nomeDaPasta
```
